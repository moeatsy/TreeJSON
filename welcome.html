About permissions

Why does Chrome say "Read your data on all websites"?

JSON APIs live on every domain on the web. To format a JSON response from any domain, Chrome requires us to declare access to that domain — there is no <json-mime-only> permission.

TreeJSON only inspects the document's content-type. If it isn't JSON, YAML, or XML, the script exits immediately. We never read browsing history. We never send any URL anywhere. We have no servers. 

// content.js — the actual detection check
if (!shouldFormat()) return;   // ← exits before touching the page
